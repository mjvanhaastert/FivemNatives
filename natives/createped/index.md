# CreatePed Domain

Ped creation and lifecycle helpers centered on `CreatePed` and related spawn/control flows.

## Scope

- Ped spawn natives for world and vehicle contexts.
- Setup helpers for mission entities and cleanup.
- Relevant ped lifecycle events and limitations.
- Runtime observations and unknowns to validate in future tests.

## Files

- `core.md`: primary ped creation natives.
- `helpers.md`: supporting helpers for ownership, persistence, and cleanup.
- `events.md`: ped-related event notes.
- `findings.md`: empirical observations and hypotheses.
